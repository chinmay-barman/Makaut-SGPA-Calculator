## Description
Makaut SGPA Calculator is a simple web-based application that helps students calculate their Semester Grade Point Average (SGPA).

Users can enter subject names, expected marks, and subject credits to instantly calculate SGPA based on the MAKAUT grading system. This tool is useful for students who want to estimate or verify their results after exams.

The project is built using HTML, CSS, and JavaScript with a clean and user-friendly interface.

---

## Features
- Add multiple subjects dynamically
- Enter expected marks
- Customizable subject names and credits
- Automatic SGPA calculation
- Delete subjects
- MAKAUT grading system overview

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## Usage
1. Add a subject by clicking **Add Subject** button
2. Enter subject name, expected marks, and subject credits
3. Repeat for all subjects
4. Click the **Calculate** button
5. The calculated SGPA will be displayed below the button
